# BIB OJT Node.js Tuto5 Update
- **URL**
```
localhost:3000 (Landing Page -> Login/register route) 

localhost:3000/register (Register form)

localhost:3000/login (login form)

localhost:3000/dashboard (Dashboard page -> can't go without login , it will redirect to login page if the user is not login yet)

We need to register first , then we need to login to see dashboard page. We can't access to dashboard page without login . It's my secret page.

```

- **Installation**
```
git clone https://github.com/AungKyawSoeOo/AungKyawSoeOo_Node.git
cd Day12_tuto5_update
npm install
npm run dev

open localhost:3000/ in the browser
```
